Rôle de cette page
Cette page centralise les demandes d’information, les questions des candidats, les messages liés au jury, aux résultats, aux certificats ou à l’organisation générale du championnat.

Contacter l’organisation

Cette page permet d’envoyer un message clair et structuré à l’organisation du Championnat de Chimie. Elle s’intègre directement au système et envoie le message dans la boîte de réception de support administration.

Envoyer un message

Le message est enregistré directement dans la boîte de réception interne lue par la page competition-support-admin.html.

Si tu es connecté, ton nom et ton email peuvent être préremplis automatiquement.

Quand utiliser cette page

Les cas les plus utiles.

Pour poser une question sur le championnat, signaler un problème d’accès, demander une précision sur les résultats, évoquer une question de certificat, ou contacter l’organisation dans un cadre plus formel.

Intégration au système

La logique retenue pour la suite.

Le formulaire envoie le message dans competitionSupportInbox, qui est justement la boîte lue et gérée par la page support administration.

Accès liés

Les pages qui complètent cette page de contact.

FAQ, règlement, résultats, vérification certificat, espace candidat, jury privé et administration forment un ensemble cohérent autour de cette page.
