Vérification officielle d’un certificat

Cette page permet de vérifier si un certificat Chem Challenge a bien été délivré et validé officiellement par l’administration. La vérification est maintenant directement liée aux certificats réellement enregistrés dans le système du concours.

Base liée au système Chem Challenge

Rechercher un certificat

Saisissez le numéro de vérification du certificat. Vous pouvez aussi essayer le code candidat s’il figure sur le document.

Formats reconnus ici : numéro de vérification admin, code candidat, ou identifiant lié au certificat.

Cette page lit désormais les certificats réellement délivrés par l’admin dans le système du concours. Tu n’as pas besoin de modifier `competition.html`, `competition-admin.html` ni `competition-jury.html`.

À propos de la vérification

Ce que cette page vérifie Elle contrôle si le certificat saisi existe réellement parmi les certificats délivrés par l’administration du concours.
Un certificat valide affiche ici
  • le nom du participant,
  • le numéro de vérification,
  • le code candidat,
  • la date de délivrance,
  • l’édition concernée,
  • l’autorité de délivrance.
Connexion avec ton système Cette page est pensée pour fonctionner avec les certificats enregistrés par `competition-admin.html` et rester cohérente avec `competition.html`, `competition-jury.html` et `insights.html`.
Rappel Si aucun résultat n’apparaît, cela signifie généralement que :
  • le code saisi est incorrect,
  • le certificat n’a pas encore été délivré,
  • ou le document ne provient pas du système officiel.
